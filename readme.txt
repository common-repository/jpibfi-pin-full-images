Extension to jQuery Pin It Button for Images that allows you to pin full-sized images provided the image is linked to its full-sized version.

== Description ==


The plugin allows you to:

* pin full-sized images instead of thumbnails if you link thumbnails to their full versions
* choose file extensions that should be considered 'full-sized' images
* choose to use full-sized images only if they come from the same domain as your website


**Please report any bugs (or feature requests) in the plugin's support forum.**

Check out the new features in the Changelog.

Please note that the plugin doesn't work if the user has JavaScript disabled.

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configuration interface can be found under `Settings` - `jQuery Pin It Button For Images` in the `Pin Full Images` tab. There's also a link to the plugin settings in the "Installed plugins" menu.
